# 常用命令

- `pnpm build`：构建 dist。
- `pnpm dev` / `pnpm dev:watch`：开发构建。
- `pnpm test`：运行 Vitest（含 typecheck）。
- `pnpm test:silent`：点状输出测试，用于 `pnpm qa`。
- `pnpm test:types`：运行类型测试。
- `pnpm typecheck`：TypeScript 检查。
- `pnpm lint` / `pnpm lint:fix`：ESLint 检查/修复。
- `pnpm biome:check`：Biome 检查。
- `pnpm qa`：本地快速门禁。
- `pnpm ci:strict`：严格门禁，含 coverage。
- `pnpm release:check`：发布前校验。
- `pnpm docs:dev`：启动 VitePress 文档站。
- Windows 常用：`Get-ChildItem`、`Get-Content`、`git status`、`rg`。
